# 📄 AI Resume Screening System

An AI-powered Resume Screening System built using **Python**, **Flask**, and **Machine Learning** that analyzes resumes, compares them with job descriptions, and calculates an ATS (Applicant Tracking System) compatibility score.

---

## 🚀 Features

- 📄 Upload Resume (PDF)
- 🎯 ATS Compatibility Score
- ✅ Matching Skills Detection
- ❌ Missing Skills Identification
- 🤖 AI Recommendation
- 👨‍💼 Recruiter Decision
- 📊 Resume Statistics
- 🌐 Responsive Flask Web Interface

---

## 🛠 Technologies Used

- Python
- Flask
- Scikit-Learn
- TF-IDF Vectorizer
- Cosine Similarity
- Pandas

- CSS
- Bootstrap
